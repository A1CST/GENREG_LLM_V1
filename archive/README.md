# Archive

Exploration artifacts from earlier iterations. Kept for historical
reference — not required to run the model.

- `reports/` — point-in-time progress write-ups (superseded by
  `../CHANGELOG.md` and `../README.md`).
- `diagnostics/` — one-off scripts used to audit specific questions
  (retrieval recall slices, chatbot behavior checks, retrieval-knob
  sweeps). Each script stands on its own if you want to re-run it.
